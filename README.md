Task 7 – Basic Sales Summary Using SQLite & Python
 Objective

Create a small SQLite database and use Python to:

Connect to the database

Run simple SQL queries

Calculate total quantity sold & total revenue

Display results using print statements

Plot a basic bar chart using Matplotlib

 Files Included

task7_sales_summary.ipynb – Main notebook with SQL + Python code

sales_data.db – SQLite database file

sales_chart.png – Bar chart screenshot

 Explanation

This task demonstrates how to use:

SQLite (built-in Python database)

SQL queries to summarize sales

Pandas to load SQL results into a DataFrame

Matplotlib to visualize product revenue

The notebook connects to sales_data.db, runs a query to get each product’s total quantity and revenue, prints the results, and generates a bar chart.
